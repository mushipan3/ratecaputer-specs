## 1. 実装目論見書

### プロジェクト概要

本ボード（A基板）は、ラテカピュータ（CAR-01規格）に準拠した自律型電源管理モジュール基板である。ATtiny202をコプロセッサとして搭載し、メインMCU（UIAPduino等）のSRAM・フラッシュ・ CPU時間を一切消費せずに、バッテリー残量監視・電源シーケンス制御・接続通知BEEPを現地完結させる。10mm × 28mmの片面実装スティック形状に全回路を収める。

### 物理仕様・製造要件

| 項目 | 仕様 |
|---|---|
| 基板サイズ | 10mm × 28mm（縦 28mmはB基板・C基板とパネライズグリッドを統一） |
| 基板厘 | 0.8mm（JLCPCB 2層 FR-4） |
| 実装方式 | Top面（表面）のみへの片面実装。裏面は全面ベタ GNDプレーン |
| 基板色 | Green またはWhite（JLCPCB標準） |

### コネクタ仕様

| CN番号 | 部品 | 機能 |
|---|---|---|
| CN1 | メインコネクタ 14ピン | UIAPduinoへのコネクタ |
| CN2 | JST SH 2ピン | バッテリー接続 |

## 2. 回路設計

### 2.1 電源制御回路

バッテリー電圧をMCP73831Tで充電し、IP5310・升降圧コンバータで安定化する。ATtiny202のI2CスレーブにMAX17048G+バッテリーゲージを接続。

```
[LiPo] -- [MCP73831T] -- [IP5310] -- 3.3V出力
                          |
                     [ATtiny202] -- I2C -- [MAX17048G+]
                          |
                         BEEP終端
```

### 2.2 バッテリー監視

- MAX17048G+のRSOCレジスタを定期読み出し
- 10%以下で警告パターンのBEEP出力
- 5%以下でシャットダウン信号

---

## 3. ファームウェア仕様（ATtiny202）

### 3.1 動作モード

| モード | 条件 | BEEP |
|---|---|---|
| 通常動作 | バッテリー >= 20% | なし |
| 残量警告 | 10% <= バッテリー < 20% | 短いBEEP（3回） |
| バッテリー監視 | バッテリー < 10% | 長いBEEP |
| 接続通知 | USB接続時 | 起動ビープ |
